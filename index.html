The 10-second executive summary

To learn more about WordPress, I did a few different small programming projects:
  • I built a test blog called Dragon's Brew Coffee Corner on the simplified wordpress.com auto-hosting site.
  • I built the same test blog using the more customizable wordpress.org installation, and installed and hosted it (on DreamHost) here.
  • I added code (using a WordPress API endpoint) to a MERN stack website that I built previously, called Dragon's Brew Cafe, to fetch blog entries from the Dragon’s Brew Coffee Corner blog (click on the “Blog” navigation bar entry on the Dragon’s Brew Cafe website above to see this in action).
  • I experimented with various mechanisms for customizing the wordpress.org version of the Dragon’s Brew Coffee Corner website.
